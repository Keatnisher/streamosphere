# streamosphere
A cloud-based file storage system focused on streaming music and video.
<br />

The backend is written in Golang with Gin for http routing and api handling, and mongo db is used along with the mongo-go-driver to interface with go. Firebase was used for user authentication and authorization.
<br />

The frontend is written in React.js, with libraries such as react-dom-router, react-media-player, and react-bootstrap for styling among others.
<br />

Webpack was used for bundling all of the javascript together, and babel for the compiliation of course.
