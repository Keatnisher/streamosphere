# streamosphere
A cloud-based file storage system focused on streaming music and video.
<br />

<b>Technologies</b>: Golang with gin, aws-sdk, and mongo-go-driver, mongodb alongside firebase, AWS EC2 and S3, and react.js.
<br />

<b>Backend</b>: Written in Golang with <i>gin</i> for http routing and api handling and the <i>mongo-go-driver</i> for interfacing with mongodb, plus the <i>AWS-SDK</i> for communication with our S3 insatnce. Mongo db was the database of choice, used in conjunction with firebase for user authentication and authorization.
<br />

<b>Frontend</b>: Written in React.js, with libraries such as <i>react-dom-router</i>, <i>react-media-player</i>, and <i>react-bootstrap</i> for styling among others.
<br />

Webpack was used for bundling all of the javascript together, and babel for the compiliation of course.
